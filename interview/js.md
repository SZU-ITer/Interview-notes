# JavaScript 高频考点梳理

## 基础

- 数据类型及其检测方法
  - 六种基础数据类型
  - 三种引用数据类型
- `this`指向
  - 如何修改`this`指向？`bind/call/apply`
- 闭包
- 防抖和节流
- `ES6`新特性
  - `let` 和 `const`（和 `var` 区别）
  - 箭头函数（与普通函数区别）
  - 解构赋值
  - `Promise`

- 执行上下文

- 异步编程
- 手写代码实现

  - 实现`bind/call/apply`

  - 实现`Symbol`
  - 实现深拷贝
